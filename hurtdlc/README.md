# HurtDLC - Minecraft Cheat

Продвинутый чит для Minecraft 1.21.4 Fabric с JNI интеграцией.

## Возможности

### Combat
- **AimAssist** - плавное наведение на игроков
  - Настраиваемая скорость
  - Дистанция и FOV
  - Легитимный режим
  
- **Triggerbot** - автоматическая атака
  - Настраиваемая задержка
  - Проверка дистанции
  - Ray-AABB пересечение

### Visual
- Красивый GUI с темной темой
- Настраиваемый цвет акцента
- Watermark с FPS

## Требования

- Minecraft 1.21.4
- Fabric Loader
- Java 21

## Установка

1. Запусти Minecraft 1.21.4 с Fabric
2. Зайди в мир или на сервер
3. Запусти `inject.bat`
4. Выбери процесс javaw.exe с большим объемом памяти

## Управление

- `INSERT` - открыть/закрыть GUI
- `DELETE` - выгрузить чит

## Технические детали

### Архитектура
- JNI интеграция для прямого доступа к Java объектам
- ClassLoader-based поиск классов (поддержка Fabric)
- Отдельное окно GUI (не overlay, нет конфликтов с OpenGL)
- Многопоточный рендеринг

### Файлы
- `hurtdlc.dll` - основная библиотека
- `500.ttf` - шрифт для GUI
- `inject.exe` - инжектор
- `inject.bat` - скрипт запуска

### Компиляция
```powershell
.\build_final.ps1
```

Требуется:
- Visual Studio 2026 (MSVC 19.50)
- Windows SDK 10.0.26100.0
- JDK 21

## Безопасность

⚠️ Использование читов нарушает правила большинства серверов и может привести к бану. Используй на свой страх и риск!
