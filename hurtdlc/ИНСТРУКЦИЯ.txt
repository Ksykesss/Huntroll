🚀 КАК ЗАПУСТИТЬ:

1. Запусти Minecraft 1.21.4 с Fabric
2. Зайди в мир или на сервер (ВАЖНО!)
3. Запусти inject_hurtdlc.bat
4. Выбери процесс javaw.exe с большим объемом памяти
5. Нажми ENTER

⌨️ УПРАВЛЕНИЕ:

  INSERT  - Открыть/закрыть GUI
  DELETE  - Выгрузить чит

🎮 ФУНКЦИИ:

Combat:
  • AimAssist - автоматическое наведение на игроков
    - Speed: скорость наведения
    - Distance: максимальная дистанция
    - FOV: угол обзора для наведения
  
  • Triggerbot - автоматическая атака при наведении
    - Delay: задержка перед атакой (мс)
    - Distance: максимальная дистанция

Visual:
  • Watermark - показывать логотип и FPS
  • Accent Color - цвет акцента интерфейса

Misc:
  • Информация о версии
  • Горячие клавиши

❗ ВАЖНО:

  • GUI - отдельное окно поверх игры (не overlay!)
  • Можно играть в полноэкранном режиме
  • Когда GUI открыто - можно кликать по нему
  • Модули работают постоянно когда включены
  • Не нужно держать кнопки для работы читов
  • GUI использует свой OpenGL контекст (не конфликтует с игрой)

═══════════════════════════════════════════════════════════

🔧 ЕСЛИ НЕ РАБОТАЕТ:

1. "Failed to find Minecraft class"
   → Проверь что версия ТОЧНО 1.21.4
   → Проверь что Fabric установлен
   → Зайди в мир перед инжектом

2. Игра крашится
   → Убедись что inject.exe выбрал правильный процесс
   → Выбирай javaw.exe с БОЛЬШИМ объемом памяти
   → Не инжекти в лаунчер!

3. GUI не появляется
   → Нажми INSERT
   → Проверь что файл 500.ttf рядом с DLL
   → Если шрифт не загрузился - будет дефолтный

4. Модули не работают
   → Включи их в GUI (галочка Enabled)
   → Проверь что ты в мире/на сервере
   → Проверь что рядом есть игроки

5. OpenGL ошибки в логах
   → Это нормально! Новая версия использует отдельное окно
   → Игра не должна крашиться
